# Convert Timestamp

**tools convert to locale datetime string**
![screenshots](./assets/imgs/screenshots.jpg)

## Features

* auto convert timestamp to date string
* call out with keyboard shortcut: `OptionOrAlt+space`, like `spotlight` in macOs
